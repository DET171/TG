  ## Installation
  ```bash
  git clone https://github.com/DET171/TG
  ```
  ## Usage (Production)
  Install the dependencies:
  ```bash
  npm i --production
  ```
  Start the project:
  ```bash
  npm run prod
  ```
  ## Usage (Development)
  Install the dependencies:
  ```bash
  npm i --production=false --no-optional
  ```
  Start the project:
  ```bash
  npm run dev
  ```
