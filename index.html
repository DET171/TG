    Prerequisites

    Node.js >= 16.3.0
    NPM >= 7.19.0

    Installation

    git clone https://github.com/DET171/TG

    After that, rename .example.env to .env. Fill in your bot token, bot prefix, and get the key and bid at Brainshop.ai

    Usage (Production)

    Install the dependencies:

    npm i --production

    Start the project:

    npm run prod

    Usage (Development)

    Install the dependencies:

    npm i --production=false --no-optional

    Start the project:

    npm run dev
